Tkibuli Coal Mine Project

2020/07-2021/08

RnD Project Design and Development. I created a full RFID payment system for Tkibuli Coal Factory including hardware and software. I designed PCB and RFID 13.56Mhz antenna. I made a simple GUI program in Python as well, that communicates with Hardware RFID card reader using Serial communication. It is basically an emulation of a shop POS computer but the payment is processed using custom RFID cards. The system is being used daily by 1000+ Employees.





Smart Solar Panel

2020/09-2021/04

Bachelor Thesis Project a.k.a. Capstone Project. Me and my team designed and built system where a Solar Panel was mounted on a plane that moved in two axis direction. The idea is that it follows the movement of the sun, which in turn maximizes the power efficiency. We used photoresistors for feedback and also added current sensors and light sensors. We transmitted all the data to the server such as power generated, bus voltage, light intensity etc. We also built an Android application where the user could view all the graphs in a simplistic way.
